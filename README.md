# aws-cloud-networking-architecture-topic2

## AWS Cloud Networking Architecture Assignment

This repository contains the Cloud (AWS) Hands-on Topic 2 assessment project focused on designing and implementing cloud networking architecture using AWS services.

## Services Used

- Amazon VPC
- Public and Private Subnets
- Route Tables
- Internet Gateway
- Security Groups
- EC2 Instances
- Network ACLs

## Topics Covered

- Cloud networking architecture
- VPC creation and configuration
- Subnet management
- Internet connectivity setup
- Routing configuration
- Security and access control
- EC2 network communication

## Steps Performed

1. Created a Virtual Private Cloud (VPC).
2. Configured public and private subnets.
3. Attached Internet Gateway to the VPC.
4. Created and configured route tables.
5. Configured security groups and network access rules.
6. Launched EC2 instances inside subnets.
7. Tested connectivity and routing configurations.
8. Verified successful network communication.
9. Uploaded documentation and screenshots to GitHub.

## Challenges Faced

- Route table association issues
- Internet connectivity problems
- Security group configuration errors
- Subnet communication setup challenges

## Outcome

Successfully completed the AWS Cloud Networking Architecture assessment by implementing and testing secure cloud networking configurations.
